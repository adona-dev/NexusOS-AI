# Ownership Handover Steps: ASync AI

## 1. Code Validation Checklist
* Verify that Tailwind CSS links inside the head render properly.
* Check input form button submit states.

## 2. Deploy Steps
* Create a Netlify or Vercel repository slot.
* Upload app.html directly (rename it to index.html).

## 3. Domain Registration
* Query Namecheap or Google Domains for: `async ai.com`.
* Set CNAME dns records to your Netlify deploy URL.
